CARRIER

変わりゆく時代や環境に適応していきたい

2015年よりWebデザイナーとして飲食事業のHP制作、人材事業で求人サービス、マーケティング事業のLP制作などでキャリアを積むと同時にデザインを言語化していく力を身につけました。

また、2017年からはSESとしてデザイン兼フロントのスキルを身につけるため様々な案件にジョインし、よりメンテナブルで拡張性の高いCSSの設計とプログラミングのスキルアップに注力しました（これによってWebデザインの設計について見直す機会を多く得ました）。

これまでの経験を生かし「そこでしか通用しない」ではなくどこへ行っても、またどんな変化に対応していける人材としてスキルアップし続けます。

アトケアプロ

design / coding / direction

アトピーの子供(小学校低学年)を持つ母親をターゲットとした健康サプリのランディングページです。
プロポリスというと渋緑色や黄色の配色のLPが多いですが、苦いイメージを持たれにくくするため、緑ではなく緑に近い黄色をベースカラーにデザインしています。こちらの商品は梱包用冊子デザインも担当いたしました。ライターのリライトやディレクターの入れ替わりなどもあり、3ヶ月近い制作時間を要しました。
